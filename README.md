# ◈ AutoDB | Sistema de Gerenciamento de Dados

Este é um projeto desenvolvido para facilitar a estruturação e o armazenamento de dados de forma dinâmica e intuitiva. O sistema permite que o usuário crie suas próprias tabelas e colunas, funcionando como um banco de dados flexível diretamente no navegador.

## 🚀 Funcionalidades
* **Criação de Schema:** Defina colunas personalizadas (Texto, Número ou Data).
* **Persistência Local:** Os dados são salvos no `localStorage` do navegador (não se perdem ao atualizar a página).
* **Busca Inteligente:** Filtro em tempo real para encontrar registros rapidamente.
* **Interface Responsiva:** Design moderno focado em usabilidade e tecnologia.

## 🛠️ Tecnologias Utilizadas
* **React.js:** Para a construção da interface dinâmica.
* **CSS3:** Estilização personalizada com variáveis e animações.
* **Git & GitHub:** Versionamento e deploy automatizado via GitHub Pages.
